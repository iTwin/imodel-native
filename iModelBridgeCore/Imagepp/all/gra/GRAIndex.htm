Dependencies

Image++ Graphic Libraries depend technically upon all Image++ Utility Libraries

 

List of Libraries

Graphic Surface (HGS)

Library containing implementation of graphic surfaces used to draw upon.

Graphic Foundations (HGF)

Graphic foundation library. All major graphic concepts are defined there.

Raster Files (HRF)

Raster File. Contains raster file definition as well as all specific format implementations for simple formats.

Raster Pixels (HRP)

Raster pixels and filtering mechanisms definitions.

Raster Foundation (HRA)

Raster foundation implementations

Images (HIM)

Imagery library. Contains implementation concerning high-level image concepts such as mosaics, virtual images, etc.

Vectors (HVE)

Vector definition including shape oriented operations

Tiff file access library (HTIFF)

TIFF low-level file format access

PictureScript (HPS)

PictureScript implementation

Image Server (HIS)

Image Server. This library contains the Internet Image Server core.

Raster Upper-Level files

Upper level raster file implementation. Contains the raster file encapsulation of PictureScript and other composite raster file implementations.

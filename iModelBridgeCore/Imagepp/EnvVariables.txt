
To build I++ with NBB and/or with DSP, we need to set these variables in your Env.

        IMAGEPP_APPS            \devipp\apps
        IMAGEPP_OPTCMP_DBG      <space>
        IMAGEPP_OPTCMP_REL      <space>


Build by image.mke

        set IMAGEPP_DLLMODE=1   To build with the configuration "DLL Debug" or "DLL Release".
        set IMAGEPP_UNICODE=1   To build the static library in UNICODE.
                                DLL confiuration doesn't support UNCODE yet.

Build options
        set HMR_EXCLUDE_DEPRECATED      to build without deprecated definition

General
   
Thumbnail Preview
Shows the thumbnail for the image file.
 

Quality Thumbnail
Increase the quality for the thumbnail preview.
 
Geo Reference
Apply the transformation matrix to the thumbnail preview.
 
Type
Displays the format and extension of the image file.
 
Location
Shows where the image file is located on your computer.
 
Size
Displays the size of the image file.
 
Uncompressed Size
Displays the size the image data would occupy if uncompressed.
 
Image Dimensions
Displays in pixels width and height of the image file.
 
Compression
Displays the compression type for the image file.
 
Color Space
Displays the color space for the image file.
 
Scanline Orientation
Displays the scanline orientation for the image file.
 
Overviews Dimensions
Displays in pixels width and height of each overviews.
 
Number of Blocks
Displays the number of blocks per width and per height for the image and its overviews.
 
Block Dimensions
Displays the block dimensions width and height for the image and its overviews.




















Resource
   
Resource Type
Displays the name of each resource of the image file.
 
Value
Displays the value of each resource of the image file.
 










Benchmark
   
Benchmark
Displays time to read, decompress, write and compress blocks (best, average worst and total)
 
Benchmark Write Blocks
Check this box to include write write to blocks in the benchmarks.
 
Send the values to the Image Insider Database
Check this box to send the results to the Image Insider database.
 
Start Benchmark
Start the benchmark for the image file.

Installation
Step 1: Copy the ImageInsider.exe file under ProgramFile\ImageInsider
Step 2: Create a shortcut for it
Step 3: Copy the shortcut under WinNT\Profiles\"Your Name"\SendTo
   
  To test the program click on an image file with the mouse right button.
 

Select: Send To > Image Insider.
